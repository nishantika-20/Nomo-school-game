# 🔥 NOMO SCHOOL

A browser-based 2D side-scrolling arcade game built from scratch using React, Vite, and HTML5 Canvas.

## 🎮 Gameplay
- Choose from 3 playable characters: Jenny, Annie, and Nico
- Collect lighters 🔥 and assignments 📄 across 5 levels
- Avoid teachers, security guards, and wardens
- Burn the evil building at the end of each level to progress
- 3 lives per level — don't get caught!

## 🕹️ Controls
- **Arrow Keys / WASD** — Move
- **Space / W / ↑** — Jump
- **R** — Retry on Game Over

## 🛠️ Built With
- React 18
- Vite
- HTML5 Canvas API
- Custom game loop & physics engine
- Pixel art sprite animation

## 🚀 Run Locally
git clone https://github.com/nishantika-20/nomo-school-game
cd nomo-school-game
npm install
npm run dev
